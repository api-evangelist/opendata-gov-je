# opendata.gov.je (opendata-gov-je)

opendata.gov.je is a open data portal open-data portal for JE running **CKAN**, exposing a standardized catalog API over ~118 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/opendata-gov-je/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=opendata-gov-je-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.8.12

## API
- **opendata.gov.je CKAN Action API** — base `https://opendata.gov.je/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://opendata.gov.je)
- Live endpoint verified: `https://opendata.gov.je/api/3/action/status_show` · ~118 datasets
